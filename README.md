## Local Setup

### Prerequisites
- Python 3.8+
- pip

### Installation

1. Clone this repository:
```bash
git clone https://github.com/YOUR_USERNAME/home-locator.git
cd home-locator
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Prepare the data:
```bash
python prepare_data.py
```

4. Run the app:
```bash
streamlit run app.py
```

The app will open in your browser at `http://localhost:8501`

## Deployment to Streamlit Cloud

1. Push this repo to GitHub
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Sign in with GitHub
4. Click "New app"
5. Select this repository
6. Set main file to `app.py`
7. Deploy!

## Updating Data

To update with the latest data:

1. **Housing**: Download latest ZHVI from Zillow
2. **Elections**: Check MIT Election Lab for updates
3. **Gun Laws**: Visit Giffords scorecard for current year
4. **Population**: Download latest from Census Bureau:
   ```
   https://www2.census.gov/programs-surveys/popest/datasets/2020-2024/counties/totals/co-est2024-alldata.csv
   ```

Then run `python prepare_data.py` to regenerate the merged dataset.
